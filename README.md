# WTWR (What to Wear?)

## Project Description

WTWR (What to Wear?) is a React-based web application that helps users decide what clothing to wear based on the current weather conditions. The application retrieves real-time weather data from the OpenWeather API and displays recommended clothing items that match the temperature and weather conditions.
Users can view weather information, browse clothing items, and interact with a responsive user interface designed to provide a seamless experience across different devices.

## Functionality

* Displays current weather data based on the user's location.
* Shows temperature and weather conditions.
* Filters clothing recommendations according to weather conditions.
* Responsive design for desktop and mobile devices.
* Dynamic weather cards with condition-specific images.
* Component-based architecture using React.
